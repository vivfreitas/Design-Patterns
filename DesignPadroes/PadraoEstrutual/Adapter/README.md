# 🦉 Adapter
Adapter é um padrão estrutural que converte a interface de uma classe em outra interface esperada pelo cliente, permitindo que classes incompatíveis trabalhem juntas.

---
## Tradução
Imagina que o seu código espere **interface A**, mas você só tem uma classe que implementa **interface B**. 

O **Adaptador** vai fazer a ponte entre os dois.
